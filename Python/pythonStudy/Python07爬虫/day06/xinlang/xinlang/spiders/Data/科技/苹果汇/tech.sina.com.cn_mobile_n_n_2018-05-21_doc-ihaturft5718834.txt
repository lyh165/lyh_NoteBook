　　新浪手机讯 5月21日上午消息，这周末英国哈里王子迎来大婚之喜，这家英国奢侈礼品生产商Goldgenie也蹭了蹭热度，推出了一款“英国王室婚礼限量版”iPhone X Elite，售价4000美元。　　这款限量版iPhone X以24K金装饰机身，背面雕刻了哈里王子与梅根·马克尔结婚照。iPhone X Elite一共限量99部，每部产品都有它的独立编号，这也是象征身份的一部分。　　除了24K机身之外，这款iPhone X Elite在参数上与普通班iPhone X没有太多出入，提供64GB和256GB存储空间，镌刻王室婚礼照的iPhone X仅限256GB版本。　　另外随产品包装盒也采用了樱桃橡木材质，并附带一对儿EarPods耳机，USB充电以及一个5W的适配器。　　iPhone X Elite皇家婚礼限量版售价为3964.16美元约合25278.7元人民，这个价格你会考虑买吗？（瑞豪）
