　　最近一段时间，各大影像厂商新机谍报集中出现。虽然并没有什么实机照片，但是从出现的各种性能配置来看还是比较详细的，可信度也比较高。新机型相对于之前的产品性能都有了较大的提升，而且几乎都将4K视频功能融入到新机中。今天我们就来给大家汇总一下，近期的新机谍报。　　　　尼康的新单反终于有了点儿眉目，据海外媒体谍报，尼康将在今年Photokina前推出新款的入门单反D3500。新相机仍然将保持在2400万像素级的照片分辨率，此外新机将加入4K视频拍摄能力，对焦系统也将有进一步的改善提升。　　据悉，新一代的尼康入门单反D3500将加入4K视频功能。此外，相机的对焦系统会从现在的11点升级到39点。机身的性能将达到现在D5600的水平，作为一款入门级单反，新机的配置还是较为吸引人的。　　：　　在去年推出D850后，尼康实际上在相机更新上鲜有动作，尤其在宣布转向专业消费市场后，入门级的产品的更新就进一步地放慢了步伐。这次D3500谍报的出现，多少对于想要购买尼康入门级单反产品的用户是个好消息。　　　　佳能G7 X Mark III预计在年内发布，现在有了一份初步参数信息。最大升级是提升了像素，从2000万像素提升到2400万像素，其他方面基本上是G7 X Mark II的加强版，升级了处理器，改进了连拍，然后加入了4K。当然实际情况如何，还要看最终发布情况。　　　　2410万像素1英寸传感器　　10fps的最高连拍速度　　DIGIC 8处理器　　24-100mm f/1.8-2.8镜头　　3英寸翻转触控屏　　支持4K 30P和1080 60P　　支持Wi-Fi/NFC和GPS　　　　新一代G7 X Mark III目前参数信息上，最大亮点就是2400万像素传感器，但是这也是可信度最低的一条。佳能新一代G7 X Mark III价格应该不会有太大涨幅，继续走高性价比路线，如果发布还是值得期待的。　　　　奥林巴斯的E-M5系列也将迎来更新了，近日我们得到了一系列关于奥林巴斯全新款E-M5 Mark III的参数谍报，新机将提升至2000万像素级别，同时支持4K视频拍摄，并且最高连拍速度可达15张/秒，一起来看看具体的谍报规格吧——　　　　2040万有效像素M43画幅传感器（与现有E-M1 Mark II同款）　　121点十字相位对焦+121点反差自动对焦　　支持5000万像素高解析力模式　　机械快门速度最高可达1/8000s，电子快门可达1/32000s　　最高连拍速度约15张/秒　　支持DCI 4K/24p、UHD 4K/30p视频录制　　3英寸104万像素翻转屏（应该支持触控）、236万像素电子取景器（1.3倍放大倍率）　　预计2018年第四季度至2019年第一季度之间发布　　　　E-M5应该算是奥林巴斯的中高阶机型，配置方面还是很厚道的。新的E-M5 Mark III今年有望问世，从参数表现上看新机在现有E-M5II的基础上带来了全面升级，像素提到了2000万的级别，并且加入了4K视频，连拍也从10张提到了15张。就纸面参数而言，E-M5 Mark III可以说是一台兼顾连拍、画质、视频的M43画幅微单，还是非常值得关注的。　　　　据FujiRumors网站消息称，富士将在本月24日召开一场新品发布会，届时将会为我们带来全新的微单机型——X-T100。据悉新机在外观上与现有的X-T20很相似，并且也将提供2400万像素级别的APS-C画幅传感器，但是价格相比X-T20将会更加亲民。　　富士目前的APS-C微单机型从造型上分为两类，一种是类旁轴的取景模式，例如X-Pro 2、X-E3、X-A5等机型，另一种则是类单反的取景模式，像X-T2、X-T20、X-H1等等。现有的X-T系列虽然推出了相对亲民的X-T20，不过整体价格上比起自家旁轴系列而言还是要高出一些，因此富士的X-T100将会是一款全新的入门定位产品，具体售价及性能参数都值得期待。　　　　富士X-T100很可能将在本月下旬发布，对于喜欢富士微单的用户而言，这款机型很可能是个门槛更低的选择。说实话之前X-T20发布的时候其实已经带来了足够多的惊喜，无论是售价、外观还是4K视频的保留，富士在产品定位上已经很有诚意，此次谍报中的X-T100若能如约而至，那么入门微单市场将再添一员虎将。　　　　理光近日接受了Imaging Resource的采访，在被问及新产品相关的问题时，理光表示将在近期发布一台宾得APS-C画幅单反新旗舰，接替现有的K-3 Mark II。另外理光还表示，新的APS-C画幅旗舰在外观上会有较大变化，与现有的K-3系列完全不同。　　除了新机的消息之外，理光在采访中还说明宾得Q系列产品线确实依然存在，并且依旧在产在售。另外理光表示以公司目前的状况来看，推出中画幅无反并不现实。　　　　结合之前的谍报以及此次的采访来看，我们很快就能看到宾得的APS-C画幅新旗舰单反了，不过从谍报来看理光宾得很可能会放弃K-3系列传统的单反外观设计，新机型的外观可能会采用类似K-1、KP的复古风格，还是值得期待的。　　　　以上就是近期各大影像厂商的新品谍报，这些新机型的性能配置都有着不错的表现。虽然目前还仅仅是谍报程度，但是可信度还都是非常高的。而且4K视频拍摄也已经成为新机型的标准配置了。
