　　　　旅行对于现代人来说，是放松心情的最佳方式，出门在外，我们能看到很多与众不同的东西，自然就需要一台趁手的相机拍照。有时候大相机太过沉重，而手机拍照总会感觉不够用，这时候你就需要一台好用的一英寸卡片机，既有着轻巧的体积让旅行毫无负担，又能帮助您拍出好看的照片。　　出门在外，用过的一英寸相机有不少，如今看来最称心如意的还是佳能G7 X Mark II和索尼黑卡RX100 IV。这两款相机都是以轻便小巧而著称，有着出色的便携性，可以让大家的旅行拍摄更加简单。　　从外观上来看，两款相机都是典型的专业卡片机设计。从上时间的使用感受来看，佳能G7 X Mark II第二代相比于第一代着重改进了机身的握持感受，特别是增加了一个舒适的小手柄，在握持操控上相对比较舒服。索尼黑卡一直延续着统一的造型设计，光面设计看上去更加具有科技感，但是也降低了手拿相机的舒适度，如果索尼能够增加一个小手柄，对使用舒适度会有不小的提升。　　机身方面，两款相机的翻转屏设计基本一致，都是向上翻转，对于旅行来说，翻转屏非常适合低角度拍摄，这样我们就不用总是弯下腰去构图取景了。在机身使用上，旅行中我最喜爱的是佳能的触控设计，佳能的触控是全功能触控，不仅支持触控对焦，而且也支持各类参数的调整。索尼黑卡RX100 IV虽然按键操作很不错，但是黑卡并没有触控，这一点很可惜。　　对于旅行来说，我们选择一英寸相机而不选择微单或者手机拍照，就是为了更加轻便，更加好用。两台一英寸相机我都用过一段时间，从便携性上说，两款相机都非常轻便，不过在小功能上，佳能G7 X Mark II的使用友好度会更好，更能够做到上手就拍照。　　　　旅行选择一英寸卡片机而不是手机，就是因为相机有着更好的画质表现，有着光学变焦镜头，可以让我们的拍摄变得更加容易。对于今天我们说到的两款相机来说，都具有2000万像素的一英寸传感器，可以拍出清晰的画面来。　　这一次去桂林出行，我同样带上了这两款相机，正所谓桂林山水甲天下，虽然这次我们在桂林没有看到一个晴天，但是阴雨天的桂林同样有着朦胧的美感。我们就来看一下两款相机在桂林的拍摄表现如何。　　这两张照片是我们在阴雨下拍摄的桂林叠彩山山顶。在阴雨天拍照，我们可以看到两款相机的色彩成像有所区别。佳能G7 X Mark II颜色偏向于冷色系，而索尼黑卡RX100 IV在这种环境下偏向于黄色。色温检测上，我更倾向于佳能G7 X Mark II的色调。　　在山顶等待日落的时候，有光线突然从云层中射出，形成了丁达尔效应。这时候我们用两天相机一起拍摄，索尼黑卡RX100 IV只能使用普通的照片模式进行拍摄，不过佳能G7 X Mark II机身具有HDR模式，因此我直接使用HDR模式进行拍摄。　　由于光线的突然到来，天空和地面形成了比较大的光比，直出照片无法显示出完整的光影动态。佳能G7 X Mark II使用HDR功能拍摄之后，天空和地面的层次感都得到了保留，而且颜色也更加丰富。从旅行游玩的角度来说，这张照片通过Wi-Fi直接上传朋友圈，就得到了很多人点赞。　　　　在实际使用中，两款相机的画质表现其实都令人满意。不过旅行中我们更希望相机的照片能够不后期就能够有好看的效果，索尼黑卡的色彩和照片明暗光比往往需要进行后期调整，我更喜欢佳能直出的色彩，基本上直接拍摄的照片就能够拍摄上传朋友圈，对于轻旅行来说，G7 X Mark II的使用体验肯定要更好一点。　　我们使用相机拍照，另外一方面就是相机具有更好的镜头焦距以及虚化效果。两款相机虽然都是一英寸的传感器，但是在镜头上有所不同。佳能G7 X Mark II搭载的是一枚等效24-100mm f/1.8-2.8的镜头，而索尼黑卡RX100 IV则是24-70mm f/1.8-2.8的镜头，从焦段上看，佳能直观感受就是比索尼更长一些。　　先说24mm，两款相机都具有24mm的广角端，因此不管是佳能G7 X Mark II还是索尼黑卡RX100 IV，在应对大风景拍摄的时候都没有什么压力，24mm下我们适合拍摄一些大场景，或者利用f/1.8的光圈拍摄虚化场景。　　其实，对于70mm和100mm来说，从数据上看不出来什么差异，但是实际上的拍照还是有一些差异的。100mm直观来说就是更长，对于旅行来说，这一点其实很重要，长一些不仅拍得更远，而且可以获得更好的虚化。　　这两张图可以直观感受到两台相机长焦端的焦距差异。可以看到使用长焦拍摄，我们可以拍到我们更想要的东西，长了30mm，对于画面构图还是很有帮助的。　　这两张图，可以看出在长焦端两枚镜头近距离虚化的差异。想要拍摄出虚化更唯美的照片，一方面可以使用更大的光圈，另一方面我们可以使用更长的焦距和离得更近拍摄。佳能G7 X Mark II拥有100mm的长焦，因此实际拍摄中能够得到更好的虚化效果。　　从一个旅行者的角度来看，不管是佳能G7 X Mark II和索尼黑卡RX100 IV都有着很好的表现，不过在很多小细节上，佳能G7 X Mark II在旅行中带来了更好的拍摄感受。更实用的焦段，更好的虚化效果，更好的操控体验，对于一个爱旅行的人来说，这些功能看似不起眼，但是却可以让我们的旅行体验变得更出色。
