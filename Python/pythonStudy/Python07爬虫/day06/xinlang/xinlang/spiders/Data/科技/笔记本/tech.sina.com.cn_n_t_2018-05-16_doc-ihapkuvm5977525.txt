　　本文来自太平洋电脑网　　窄边框设计是未来发展的一种趋势。不仅可以提升笔记本电脑的整体美感，而且还能让获得更宽广的视野，拥有更加逼真的使用体验，所以现在很多笔记本厂商都在窄边框设计方面下了十足的功夫，戴尔在窄边框设计上算是起步较早的，全新的XPS 13汇集了这些年戴尔在窄边框设计上的深度思考，给人的感觉更加优雅成熟，足够惊艳全场。同时全新的戴尔XPS 13也依然继承了便携身形、精致做工等之前产品的优点，性能方面也足够给力。　　　　　　全新的戴尔XPS 13屏幕边框已经窄至4mm，相比于上一代产品，它的屏幕宽度约窄了23%，现在很少有笔记本可以将屏幕边框做成这样的宽度，就目前看来，这个宽度已经算是窄边框设计中极致。屏幕边框更窄，就意味着戴尔XPS可以拥有更高的屏占比，它的屏占比已经高达80.7%，更大限度地拓宽用户的视野，模糊了屏幕的边界，让用户在虚拟和现实之间更加来去自由。　　摄像头的放置问题一直是很多窄边框笔记本生产厂商思考的问题，戴尔XPS 13摄像头把原本放置在上边框的摄像头挪到了下边框，放在了和logo相邻的位置，这个设计可以说是非常的巧妙，节约空间的同时而又不失美感。　　　　想要打造出极致的视觉体验，窄边框设计仅仅是第一步，屏幕素质也必须要足够好。全新的戴尔XPS 13屏幕分辨率是3840×2160，已经达到了4K级别，图像显示更加逼真细腻。另外戴尔XPS 13的亮度也已经高达400尼特，对比度为1500：1，sRGB色域已经达到了100%，现在很多窄边框笔记本的屏幕素质都没有达到如此之高，可见这款戴尔XPS 13的在屏幕设计方面非常之走心。　　　　全新的戴尔XPS 13的身形依旧非常轻巧灵动。它的裸机重量仅为1.21kg，机身最薄处仅为7.8mm，机身最厚处也只不过11.6mm，它的长宽为302mm×119mm，整体体积比上一代产品缩小了24%。无论和现在主流的轻薄本相比，还是和它的上一代产品相比，它的便携性都有了不小提升。　　除了身形上足够便携之外，全新的戴尔XPS 13在配色和做工上也足够大胆。它的A面配色采用的是时下比较风靡的玫瑰金配色，并且采用磨砂工艺制作而成，显示出戴尔XPS 13典雅尊贵的一面，而且手感更是精致细腻。　　而全新的戴尔XPS 13的C面配色选择则是纯白配色，又显示出它简洁素雅的一面。这里我们着重介绍C面的材质，C面采用了全球首创的白色硅纤维制作而成，创新就能编制技术，让整个C面更加坚韧还更加轻薄。除此之外，C面还覆盖了Velvecron防污涂层，防污效果非常好，让C面始终都洁白如初。戴尔XPS 13键盘的键程和反馈也设计的非常合理，可以让用户充分享受输入的快感。　　　　为了可以让戴尔XPS 13拥有更加美观便携的身形，它为用户提供了3个Type-C接口和1个SD卡槽，非常地节约空间，但却可以满足用户地大部分需求。Type-C接口可以连接诸多的存储设备和拓展设备，所以可以基本满足常规的使用场景。而且这些接口还是雷电3接口，数据传输得非常迅速稳定，而且还能为机身充电，功能非常全面。　　　　首先我们先来这台戴尔XPS 13的整体配置。全新的戴尔XPS 13搭载了Intel Core i7-8550U处理器，拥有16GB的DDR3内存，拥有1TB的NVMe SSD。相比于现在的主流轻薄本来说，戴尔XPS 13的配置上升了一个台阶，这就让有能力处理器更高负载的任务。　　　　根据CPU-Z检测的数据来看，Intel Core i7-8550U处理器拥有4核心8线程，工作主频1.8GHz，睿频最高可加速至4.0GHz，热设计功耗为15W，三级缓存为8MB，采用的是14nm制作工艺，拥有Kaby Lake架构。相比于轻薄本入门款的Intel Core i5-8250U处理器来说，Intel Core i7-8550U的主频和睿频都有所增加，这让戴尔XPS 13可以胜任更加复杂的工作场景，同时还能节约能耗。　　　　CINEBENCH R15是一款测试处理器性能的软件，它通过渲染一张高精度的图片，最后给出一个处理器的综合得分和排名。在CINEBENCH R15基准测试中，Intel Core i7-8550U的多核处理能力得分为582cb，单核处理能力得分为155cb。相比之下，Intel Core i5-8250U处理器的多核处理能力的分数为582cb，单核处理能力的分数为155cb。这样看来，Intel Core i7-8550U处理器的确将戴尔XPS 13的性能提升了不少。　　　　在显卡方面，戴尔XPS 13的图形能力主要依赖于Intel Core i7-8550U处理器上边的核心显卡，显卡型号为Intel UHD Graphics 620，这款核心显卡可以保障戴尔XPS 13完成常规任务。　　　　3DMark是一款测试计算机的图形处理能力的软件。根据每台计算机的性能，这款软件会有不同的测试模式。测试戴尔XPS 13，我们选择了最适合它的Sky Diver模式，这个模式主要被用于中端手提电脑。在这个模式下，戴尔XPS 13的测试成绩为4936分，说明它的图形处理能力这方面表现还是不错的。而入门配置的轻薄本在这个模式下的得分大概是3400分左右，戴尔XPS 13的图形处理能力比他们强不少。　　　　固态硬盘现在可以说是笔记本的标配了，但是容量不足、成本高这些问题却一直困扰着很多用户。为了让用户享受到比较舒适的使用体验，戴尔XPS 13在硬盘方面的升级也是非常给力的，它提供的是1TB容量的固态硬盘，容量非常充裕。同时它的硬盘采用的是NVMe的传输协议规范，拥有非常稳定和高效的传输速度。　　在AS SSD Benchmark的硬盘读写专项测试中：戴尔XPS 13的持续读取速度为2621MB/s，持续写入速度为315MB/s；4K随机读取速度为933MB/s，4K随机写入速度为92MB/s；硬盘性能总体得分为1872分。这样的测试数据再次证明这块硬盘的强劲性能。　　　　PCMark 10是一款测试计算机综合性能的软件，测试项目包括存储、计算、游戏、图形处理能一些我们常用于的功能，所以参考价值也比较大。戴尔XPS 13的得分为3598分，其中常用基本功能得分为7952，生产力得分为5945分，数位创作得分为2675分。这样看来，戴尔XPS 13的整体性能还是不错的，可以满足办公家用所需。相比之下，同类型搭载入门的配置的轻薄本在这个软件里跑分大概3100分左右。　　　　轻薄本之所以做得更加便携， 一个重要的目的就是为了更适合出差旅行，所以续航能力也必须要跟得上。戴尔XPS 13续航能力也足够强劲，根据AIDA64 Extreme Edition检测出的电池数据，戴尔XPS的电池容量为51999mWh，比目前主流轻薄本的电量高出了不少。　　而在PCMark 8续航能力专项测试中，我们将戴尔XPS 13的屏幕亮度调到了50%，它最后的续航得分为4小时18分钟，这样的成绩算是比较优异了。在同等测试条件下，很多轻薄本的续航成绩都在4小时以下。　　　　　　　　全新的戴尔XPS 13在众多细节方面都进行了强有力的升级和极致的创新。首先说屏幕边框宽度只有4mm，这样的宽度是很多笔记本无法企及的。4K分辨率、400尼特亮度、1500：1对比度，如此高度值的屏幕现在也是比较少见。同时依旧保持着轻薄的身形和精致的做工。另外，Intel i7-8550U处理器、16GB的运行内存、1TB超大容量的固态硬盘，这样强劲配置足够可以让戴尔XPS 13应对更多复杂的工作环境。如果你想追求品质生活，同时又对性能有一定追求，全新的戴尔XPS 13是你不错的选择。
