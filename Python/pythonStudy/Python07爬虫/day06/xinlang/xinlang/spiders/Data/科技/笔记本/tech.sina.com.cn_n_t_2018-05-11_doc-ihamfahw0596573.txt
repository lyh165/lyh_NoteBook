　　本文来自泡泡网　　相信很多玩家对于戴尔旗下的游戏笔记本一定不会陌生，“游匣”这个称呼已经深入人心了，不过在前一段时间，戴尔对游戏笔记本的名称进行了重新规划，取而代之的是戴尔全新的G3、G5和G7系列。　　虽然名称的更换对于玩家来说，更容易进行区分，但需要注意的是，G7并不代表顶级旗舰型号，G3也并非是性价比的代表，三个系列的产品都有着自己的特点和不同。我们近日就拿到了一款戴尔G7游戏本，下面就一起来看看它的表现如何。　　　　　　从外观就能看出这款产品的特别之处，戴尔在G7游戏本中首次加入了白色的配色，在众多游戏笔记本中，白色并不多见，整体看上去非常干净整洁，机身A面仅有中央部位的LOGO，它采用了PET材质真空镀膜打造，在光线的照射下，可以反射出不同颜色，对整体的颜值有着极大的提升。全新的LOGO工艺再加上白色外观，相信这款游戏本的推出，对于追求颜值的游戏玩家来说，有着不小的诱惑力。　　戴尔G7游戏本在背面与此前的游匣系列没有太大的差别，上下两条防滑脚垫和大面积的进风孔，依旧是只有一颗螺丝进行固定，拆机升级会非常方便。　　较为遗憾的是戴尔G7游戏本并没有采用时下流行的窄边框元素，转轴部分沿用了游匣系列的设计，笔记本C面采用了金属材质，手感细腻，质感出色。可以发现这款G7游戏本整体有很多蓝色的原色，B面的LOGO、键盘字母颜色和触摸板边框都采用了蓝色的配色，与白色的外观非常搭配。　　在笔记本后方出门口位置的两侧，增加了蓝色格栅，也将颜值提升了不少，中间偏左的位置配有“G7”的金属铭牌，有很高的辨识度。　　　　戴尔G7游戏本使用了一块15.6英寸的防眩光 IPS 屏幕，分辨率为 1920 x 1080，这已经是当下游戏本的标配了，下面我们使用红蜘蛛5专业较色仪对屏幕进行具体的测试。　　测试结果显示，戴尔G7游戏本的屏幕覆盖了 64% 的 sRGB 色域，46% 的 NTSC 色域，表现正常。　　　　在键盘方面，戴尔G7游戏本延续了游匣系列的优秀水平，1.2mm的键程带给用户出色的敲击体验，按键的回弹和反馈迅速，对“WASD”四个高频按键进行了独立标记，很适合用来打游戏，笔记本设有小键盘区域，使用中更加方便多种操作。　　戴尔G7游戏本将指纹识别和电源按键进行了融合，能够以最快速度进行系统，对于安全性也有着一定提升。　　戴尔G7游戏本拥有两级亮度可调的蓝色背光，在夜晚使用更加轻松，蓝色的光线非常柔和，看上去十分舒适，在黑暗环境下有着很高的颜值。　　　　戴尔G7游戏本在接口方面非常丰富，机身左侧分别有1个耳机麦克风插口，2个一代 USB 3.1 接口，1个 Type-C 接口（支持雷电3）和一个 HDMI 2.0 接口。　　机身左侧分别是电源接口、网线接口、一代USB 3.1 接口和一个SD卡槽，戴尔G7的接口不仅丰富，而且扩展性强大，有了雷电3接口的加入，对于追求多屏的用户来说，可以更好地支持；HDMI 2.0 有着更大的带宽，3个 USB 口足够日常外接使用。　　在重量方面，戴尔G7游戏本裸机为2.8kg，旅行重量为3.5kg，不太适合经常外出携带使用。　　　　戴尔G7游戏本搭载了 Intel Core i7-8750H处理器，GTX 1060 Max-Q 显卡，16GB内存，256GB固态硬盘和1TB机械硬盘，配置强大，下面我们进行性能方面的测试。　　　　戴尔G7游戏本所搭载的 Core i7-8750H 首次将移动平台的消费级处理器带到了6核心，处理器为6核12线程设计，功耗为45W，主频2.2GHz，睿频4.1GHz，9MB的三级缓存。　　我们使用 CineBench R15 对处理器性能进行测试，单线程分数为169cb，多线程分数为1118cb，性能强大，在增加了两个核心后，相比于上一代的处理器提升非常明显。　　　　戴尔G7游戏本采用了一块 GTX 1060 6GB 显卡，不过这款显示采用的是 Max-Q 方案，在性能上相比“完整版”会有一定的差距，显卡核心频率为1265MHz，显存频率为2002MHz。　　我们使用 3DMark 的 Fire Strike Extreme 模式进行测试，最终的图形分数为5120分，表现不错。　　　　戴尔G7游戏本采用了一块256GB的海力士固态硬盘，我们使用 AS SSD 进行速度测试，最终连续读取速度为501.16MB/s，连续写入速度为461.53MB/s，表现正常。　　　　我们使用 AIDA 64 和 Furmark 对处理器和显卡进行双烤，满载运行20分钟后，我们使用热成像仪来记录笔记本的表面温度，笔记本正面的最高温度不到50°，中间靠近转轴有一小部分热量堆积的区域，不过不会影响游戏体验，笔记本背面几乎没有高温堆积的区域。　　虽然处理器增加了两个核心，但总体温度相比此前的游匣系列控制得更好，这样的散热水平在游戏本中属于非常不错的成绩了，戴尔G7游戏本在散热设计上做了全新的优化，从温度的表现上就能够显现出来。　　戴尔G7游戏本在温度上的控制非常不错，但是在性能方面也丝毫没有打折扣，满载烤机20分钟后，处理器的功耗依旧维持在45W左右，没有进行降频，显卡的频率也保持在1200MHz左右，能够保证以较强的性能长时间进行游戏，综合来看，戴尔G7游戏本在散热方面的水平是非常不错的。　　　　近一年多来火爆的《绝地求生》可以称得上是游戏PC的试金石，由于优化较差，因此在这款游戏中有上佳表现的产品，都有着不俗的性能，这款游戏甚至已经成为了游戏PC必测的项目之一，我们同样也使用这款游戏来进行帧率测试，在游戏中，我们挑选场景较为复杂，战斗激烈的机场进行测试。　　我们使用 Fraps 对游戏进行100秒的帧率记录，由于场景复杂和交战频发的原因，FPS数值会有较大的波动，属于正常现象，我们在高画质的设置下，100秒内的最高帧数可达119FPS，平均帧数有81FPS，可以非常流畅得在高画质下进行游戏，性能出色。　　　　戴尔G7游戏本依旧使用了游匣系列的模具，延续了优秀的做工，机身内部采用了防滚架设计，整体看上去干净整洁，没有凌乱的排线和胶布。用户可以对2.5英寸机械硬盘（或固态硬盘），2280规格的 M.2 固态硬盘，无线网卡和内存条进行更换升级。　　　　戴尔G7游戏本的特点非常鲜明，主打的两个特征就是高颜值和强大性能，虽然在它身上还能看到不少游匣系列的影子，但是在细节方面戴尔都进行了更精进的优化，整体的使用体验也更加出色；白色加蓝色的设计足以让追求颜值的玩家心动，而强大的性能则是游戏玩家从始至终的追求。如果你想选择一款颜值超高，性能强大的游戏本，戴尔G7就是一个最好的选择。
