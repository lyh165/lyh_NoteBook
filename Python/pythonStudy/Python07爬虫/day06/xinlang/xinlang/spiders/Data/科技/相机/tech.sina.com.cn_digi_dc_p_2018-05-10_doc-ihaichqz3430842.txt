　　本文来自佳友在线　　不知从何时其，无反相机流行起了多旗舰策略。作为发布世界第一台无反相机的松下，也在2017年年底及2018年年初先后发布了G9和GH5S，与之前的GH5一起组成了现在松下无反相机的旗舰“三剑客”。“摄影摄像”混合旗舰GH5、“摄影”新旗舰G9以及本篇的主角，松下最新的“摄像”旗舰机GH5S。　　作为新的“摄像”旗舰机，松下GH5视频记录功能进一步扩展，其是世界上第一台可录制Cinema 4K 60p / 50p的无反相机，并在低光照条件下拥有更好的图像质量。 这款为专业电影级视频工作者设计和开发的GH5S，实现了LUMIX相机历史上最高的ISO灵敏度和图像质量。　　GH5S应用了双原生ISO技术和维纳斯引擎的全新1020万像素Live MOS传感器， 可实现最高ISO 51200的高灵敏度记录，忠实再现图像中的暗部细节。该传感器还支持14位RAW格式的照片拍摄，为专业RAW格式的工作流程开发提供更高的灵活性。　　视频录制上，松下GH5S的规格强大，是第一台可录制Cinema 4K 60p / 50p的无反相机，GH5S还可以在Cinema 4K 24p和4K 30p / 25p / 24p下录制4：2：2 10bit 400-Mbps All-Intra，以及在全高清格式下录制200-Mbps All-Intra。 而且4K、全高清视频录制都没有时间限制。另外，GH5S在照片风格下还拥有混合对数伽玛（HLG）4K HDR视频功能。而在GH4和GH5中需要另外购买的V-LogL也已在GH5S中预先安装，用户无需再单独购买软件升级密钥。　　　　　　在机身设计上，GH5S直接沿用了GH5的设计，两者在机身尺寸上也完全一致。这看似偷懒的设计，实则对用户而言是有百利而无一害的，相同的尺寸和外型设计意味着GH5S能完美兼容GH5的各种配件，诸如潜水外壳之类的配件。　　松下GH5S的正面与GH5完全一致，区别在于型号的标识上，多了个S，松下也采用了醒目的红色，让用户能一眼区分。而且还在左肩部的拍摄模式拨盘最下方用了一圈红色。　　机身背面，GH5S也与GH5完全一致，习惯了GH5的用户也能直接无缝过渡到GH5S的使用上。　　机身顶部，松下将GH5上的摄像红色小圆点键改为了全红的录制键，看上去是不是很专业呢？　　GH5S底部设计还是很常规的，只有电池仓、外接手柄端子和一个三角架安装孔。　　机身两则主要是些输出端口仓以及GH5S的双SD存储卡槽。　　　　松下GH5S采用了一块1020万有效像素Live MOS 传感器和用于面向专业人士的松下摄影机Varicam使用双原生ISO技术以及最新的维纳斯引擎，由此获得了高灵敏度视频记录中的高画质。双原生 ISO是一种可以无缝地连接两个独立原生ISO电路的技术，其通过在增益处理之前根据灵敏度选择最佳电路来最大程度地降低噪点的产生。 因此，它允许最大ISO 51200的高灵敏度记录。 这种双原生ISO技术可以在LOW（ISO160-800）和HIGH（ISO800-51200）之间手动切换。。　　GH5S配备了一个大型LVF，具有惊人的高倍率，约为1.52x / 0.76x（相当于35mm相机）。高精度，高速OLED显示器具有368万像素高分辨率和100%视野。用于LVF的OLED采用120fps的高速响应，最小时间滞后小于0.005秒，画面流畅。　　松下GH5S的显示屏采用静态型触摸控制系统，3.2英寸，可自由翻转，162万像素的高分辨率，3：2的比例，视场约100%。屏幕支持上下倾斜大约270度，这使得在较高或较低角度的拍摄取景更加容易，侧面翻转的方式也避免了安装在三脚架上使用的不便。GH5S还拥有夜间模式，可以提供温和的背光。　　位于GH5S机身正前方的闪光同步端口，同时在GH5S上，该端口还支持BNC转换数据线，兼容时间码IN / OUT，可与外置设备同步时间码，很容易对多台摄像机拍摄的影片进行非线性编辑。　　位于机身左侧手柄上方的遥控接口。　　位于GH5S机身右侧的外接麦克风接口、外接耳机接口、HDMI接口和USB数据接口。从接口形状可以看出，GH5S的USB数据接口采用的是USB3.0标准，这将大幅增加其数据传输的速度，同时该接口还支持USB充电，能方便的延长整机的续航时间。　　GH5S配有兼容高速，高容量UHS-II和视频速度等级为60的双SD卡槽。用户可以灵活选择接替记录，备份记录或分配记录等记录方式。　　GH5S采用了一块型号为DMW-BLF19GK的1860mAh锂电池作为电源。续航能力最大约为440张左右，相比GH5表现有所进步。　　　　作为新的视频旗舰，松下GH5S在视频录制方面的规格可谓强大。首次在无反相机上实现了Cinema 4K（4096x2160）60p / 50p视频记录，GH5S还可在Cinema 4K 24p和4K 30p / 25p / 24p格式下录制4：2：2 10bit 400-Mbps All-Intra，以及在全高清格式下录制200-Mbps All-Intra。 而且4K、全高清视频录制都没有时间限制。　　值得称道的是，本次松下直接在GH5S中预装了V-LogL，通过Log记录为后期制作过程中的色彩分级提供了更多的灵活性和更大的动态范围，V-LogL具有12档动态范围。LUT（查找表）与Rec.709标准在摄像机默认情况下是可用的，用户也可以自定义LUT的VLT格式。　　同时，GH5S支持带有LUT的V-LogL视频片段的实时观看。这样的好处就是在拍摄时用户可以按照色彩校正过的视频画面来调整曝光，让V-LogL视频的曝光更准确。这是由于V-LogL的原片画面是灰灰的，只看这个画面来调整曝光是需要一点经验的。样片中只是简单的在V-LogL原片上套用了一个松下官方的LUT，但已经是可以看到直观的变化。淡然，你也可以自行根据实际需要来调色。　　为了进一步提升视频表现力， GH5S也提供了各种效果。 VFR（可变帧速率）可让用户以C4K / 4K（60 fps，最大速度为24p的2.5倍）和FHD（240 fps，最大速度为24p的10倍速度）录制升格/降格视频。快动作视频的最低帧频是2 fps。　　此外，LUMIX GH5S可实现4K HDR视频。 HDR（高动态范围）是一种在图像中重现明亮部分和黑暗部分细节的模式，使其看起来就像人类的眼睛一样。摄像机使用与ITU-R BT.2100兼容的指定伽玛曲线记录视频，可以在照片风格中选择混合记录伽玛（HLG）。不过HDR视频的观看需要显示设备的支持，好在在电视机方面，HDR已经在前两年开始普及了起来，人们已经可以很方便的在家就欣赏到HDR视频带来的色彩感官体验。　　　　松下GH5S搭载1020万有效像素Live MOS 传感器和维纳斯引擎，采用双原生ISO技术从ISO160-51200， 可扩展至ISO100–204800。下面我们就来看下实际的感光度表现。　　从测试图片可看出，松下GH5S在ISO 800以内都能够保持相对干净的画面，细节保留完整，上升到ISO 1600时出现少量人眼可见的噪点，直到ISO 3200开始噪点增多细节开始流失，涂抹感逐渐增大。　　虽然静态方面，GH5S的高感表现一般，但在动态方面就完全不一样了。　　动态方面我们之选取了ISO 400和ISO 2500这两档的原生ISO。高感表现看视频非常明显，ISO 2500下的画面非常干净，与静态同档ISO表现完全不同。不愧是为视频而生的机器。　　　　　　　　松下GH5S所以的改变和功能的全规划都将其指向无反相机专业视频机，其拍摄指标是当今无反相机中的NO.1，双原生ISO、预装V-LogL、支持HLG、无限时录像等等，都是专业视频人员所需要的。因此，完全有理由相信GH5S会像其前辈GH4、GH5那样成为众多专业视频拍摄人员的选择。这点上松下也通过完全沿用GH5的机身设计来让GH5S能完全兼容前代的配件，这对于之前以GH5为主力机种的专业视频人员能无缝过渡到GH5S的使用上，而无须再另配相关附件，成本上可是能节约很多的。　　从松下新的“三剑客”期间战略也可以看出，松下也意识到了自身无反相机系统的劣势，因此通过专机专精来提升产品竞争力。尤其是这次GH5S在松下原本广受好评的视频领域再次进步，这无疑会让许多原本的用户甘心掏腰包升级，同时也相信会吸引到许多潜在用户加入到松下的无反相机系统中来。　　　　1020万高感光MOS　　双原生ISO　　C4K
