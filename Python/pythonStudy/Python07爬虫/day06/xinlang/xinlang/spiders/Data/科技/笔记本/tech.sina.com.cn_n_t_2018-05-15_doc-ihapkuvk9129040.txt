　　游戏本在玩家的印象里总是傻大黑粗，粗犷的外观，突破天际的厚度以及狂暴的散热都是大家对传统游戏本的固有印象，IT之家之前也测试了微星搭载Intel Core i9处理器的GT75 Titan 8RF，其硕大的机身，直升机般的风扇声音以及暴躁的散热让人印象深刻。不过绝大部分的消费者尤其是经常外出的游戏爱好者基本上与这种准台式机无缘了，那么有没有游戏笔记本既拥有出色的性能，同时在重量和厚度上没有做太大的妥协呢？　　Nvidia在2017年正式推出了Max-Q版的10系显卡，极大程度地缩小了游戏本原来庞大的体积，让游戏本也能朝着轻薄化发展，而去年我们也看到了许多全新形态的游戏本，这些本子没有庞大的身躯，变得更加便携，同时性能相比较传统游戏本缩水并不是很严重，而这些游戏本也成为了市场上的香饽饽。　　2018年4月3日，Intel正式发布的第八代酷睿处理器，相比较第七代酷睿处理器性能更加出色，与此同时，OEM们也同一时间推出了自家的搭载第八代酷睿处理器的笔记本，这些笔记本不但采用了最新的硬件，更是采用了超窄边框屏幕，增加了笔记本的整体颜值，同时基于Nvidia Max-Q技术又可以让笔记本的体积与重量下降许多。　　目前IT之家就拿到了一款基于Max-Q技术以及Intel第八代酷睿处理器的技嘉AERO 15X v8笔记本，和其他游戏本相比，技嘉的这款笔记本更加地轻薄，就让我们一起来看看这款号称既保证颜值又拥有不俗性能的轻薄游戏本吧。　　　　我们先来看下技嘉Aero 15X v8笔记本的总体外观，技嘉Aero 15X v8笔记本的机身外型设计与用料材质上，与去年推出的AERO 15系列游戏本相同，顶盖采用铝合金材质打造，顶盖上的Logo自带白色LED背光。　　紧接着我们看下Aero 15X v8的正面设计，由于采用的是Max-Q技术的显示技术，Aero 15X v8的正面十分地轻薄，同时没有任何的外接接口。Aero 15X v8的尾部并没有额外的接口，因为无论是电源还是外接接口，技嘉Aero 15X v8都将其放置在左右两侧。　　技嘉Aero 15X v8机身厚度约为18.9mm，机身左侧配备了RJ-45接口、USB3.1、HDMI 2.0、mini DisplayPort 1.4、耳机/麦克风二合一接口。而技嘉Aero 15X v8的右侧配备的是Type-C接口、2个USB3.1接口。　　然后我们看下笔记本的后侧，技嘉Aero 15X v8的机身底部采用一体式底盖设计，在主要硬件和散热组件的位置采用了大面积的通风窗，加大进风量有助于提升整机的散热。　　技嘉采用的是三面窄边框设计的15.6英寸IPS显示屏，单侧屏框宽度为5mm，极大地增强笔记本的观感。同时分辨率为1920 x 1080，并拥有144Hz刷新率的刷新率。技嘉Aero 15X v8将摄像头设置在屏幕下方，看起来比较尴尬，同时笔记本还拥有光线感应器，让屏幕根据环境光线自动更换亮度。　　技嘉Aero 15X v8的键盘则是采用孤岛式键盘配置，键程为1.7mm、键距则为18.5mm，敲击手感偏重回馈感算是明显。另外键盘支持全彩背光功能，并可以通过自定义每一个键位的背光颜色。　　总的来说技嘉Aero 15X v8给人最大的特点就是轻薄，和传统游戏本粗犷的外观相比较显得比较娇小，适合用户外出携带，同时超窄边框的加入也让整个笔记本的颜值加不少分。　　最后我们看一下包装。Aero 15X v8在包装箱上做了小小的改变，除了使用环保可回收材质外，并且技嘉对“AERO”品牌的LOGO也进行了重新优化设计，整体的视觉感似乎会更好一点。　　　　作为一款轻薄游戏本，游戏本是技嘉AERO 15X v8的基本属性，因此在配置上技嘉AERO 15X v8并不会妥协太多。技嘉搭载的是Intel Core i7-8750H处理器，16GB内存以及512GB PCI-E SSD，同时配备了Nvidia GTX 1070 Max-Q显卡，目的自然是在轻薄机身下取得令人满意的成绩，那么技嘉AERO 15X v8的实际表现究竟如何呢？我们还是用成绩来说话。　　　　Cinebench r15中文版是一款基于Cinem4D引擎的处理器测试软件，它可以同时测试处理器子系统、内存子系统以及显示子系统。和大多数工业设计软件一样，CineBench可以完善地支持多核/多处理器，它的显示子系统测试基于OpenGL。　　技嘉AERO 15X v8笔记本所搭载的Intel Core i7-8750H在CineBench R15的得分为1162分，相比较上一代处理器性能提升巨大，这个成绩已经和桌面版处理器不相上下，甚至比上一代的桌面处理器的性能更加出色。　　　　在CPU-Z的测试中也可以看到，Intel Core i7-8750H尽管基准频率只有2.2GHz，但是正常运行时睿频才是王道，于是测试时频率时常跳跃至3.8-4.0GHz，和Intel Core i7-7700K相比多线程性能处于绝对的领先，而单线程由于频率的原因落后大约15%。　　　　对于如今的笔记本来说，瘦长的M.2 SSD似乎成为了高端笔记本的标配，同时这些SSD性能不俗，给任以惊艳的表现。在CrystalDiskMark的测试中，技嘉AERO 15X v8笔记本取得了2.4GB/S的读取速度以及1.586GB/S的写入速度，同时4K性能也是极其出色，当然这已经是目前PCI-E SSD的常规操作了。　　　　紧接着我们来看下Nvidia GTX 1070 Max-Q的图形性能。在理论测试中取得了6991的图形分，比GTX 1070低了大约15%，当然这也和Max-Q版本的GTX 1070显卡的频率有关系。　　在实际游戏测试中，《孤岛惊魂5》全特效能够取得63帧的成绩，在《刺客信条：起源》的平均帧率为39帧，至于《守望先锋》能够在中高特效下取得144帧的成绩。　　毕竟这是一款基于GTX 1070核心打造的显卡，底子还是在那里的，尽管Max-Q技术让笔记本的性能有却缺少，但是面对1080P分辨率的游戏还是绰绰有余。　　　　既然是一款游戏笔记本，我们还是要看看这款笔记本的发热以及噪音，毕竟技嘉AERO 15X v8笔记本定位于超薄笔记本，能不能压住Intel Core i7处理器还是需要进行测试。　　我们通过运行3DMark、Furmark等拷机软件以及AIDA 64和软媒魔方等工具来查看技嘉AERO 15X v8的温度。　　在实际测试中，技嘉AERO 15X v8笔记本对于CPU温度并不能压制到一个比较满意的水平，85-90度的平均温度显然不太理想，不过考虑到技嘉AERO 15X v8笔记本是一款轻薄游戏本，其散热性能自然不能和普通厚度的游戏本相比较，而且尽管CPU的温度处于高烧状态，但是GPU的温度却保持在75度左右，同时技嘉AERO 15X v8笔记本也没有做太大的降频处理，实际频率在3.4-3.6GHz左右徘徊。　　噪音方面，技嘉AERO 15X v8笔记本的待机声响为46分贝，而满载声响达到了60.1分贝[1]，46分贝大约为办公室办公的声音，而60分贝和人正常交谈的声音相差不大，这属于正常水平，毕竟轻薄本带来的是更加严苛的散热条件。　　　　得益于Nvidia Max-Q技术，目前的轻薄游戏本性能并不落伍，即使比标准版显卡性能略微落后，但是GTX 1070的底子还是存在的，面对1080P分辨率的游戏并不虚，也能完成在144FPS下运行《守望先锋》的任务，毕竟现在笔记本面对4K分辨率还是有点压力。　　既然性能已经没有太大的短板，同时温度尚可忍受，轻薄游戏本的优势就已经显现出来了，无论是轻薄的外观还是重量，面对传统游戏本，这些笔记本自然经常出差的游戏爱好者的青睐，同时超窄边框的运用也让技嘉AERO 15X v8笔记本的颜值大大提升。　　只不过阻碍普通消费者购买这些游戏本的一大因素恐怕是成本了，因此和传统游戏本相比，这类轻薄游戏本的价格普遍偏高，例如技嘉AERO 15X v8入门版的售价也超过了1.2万元，主流游戏爱好者似乎很难接受这个价位。　　当然技嘉AERO 15X v8笔记本等此类游戏本的存在就是告诉人们原来游戏本也能如此轻薄，并且颜值也不差，至于购买人群，这个世界上有大把不差钱的消费者，就让他们去购买吧。
